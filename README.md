Mui简历
===

Mui简历是一个新形态的简历工具。用户不再需要像往常一样填写复杂冗长的个人信息，而是通过跟AI就业辅导对话，创造出一份非常优秀的简历。
Mui简历的目标是让用户在不是很有写简历的经验的前提下，也能够写出足够优秀的简历。
同时我们还希望对所有的求职者给予就业辅导，让大家都能够在面试阶段减少不必要的非战斗减员。


界面
---

- 以 Chatbot 对话框的形式为主
- 自动记录关键信息，实时生成简历
- 仅当对话导致个人信息发生变化时，才会生成一个新的简历版本（默认保留最近 10 个，可配置）
- 用户可以创建多个对话，以便产生不同的简历。比如我有一份简历是投给心仪的公司的，还有一份简历是海投给其他所有公司的。还有一份简历是我在论坛上发给其他人，请其他人点评的脱敏简历等等


技术栈
---

- Cloudflare Worker + D1 + KV + R2 + 其它需要的 Cloudflare 基础设施
- Next + OpenNext
- TypeScript
- Lucide
- Zustand
- TailwindCSS
- AI provider：兼容 OpenAI API 的服务，和 Google Gemini


项目架构
---

packages/
  - app 应用主体
  - shared 公共组件
  - ui UI 组件
  - cron 所需的定时任务
  - website 网站
